# Oficina-Automax
  Gerenciamento de uma oficina mecânica em SQL
  Este repositório contém o desenvolvimento completo de um banco de dados relacional 
baseado no mini-mundo de uma Oficina Mecânica.
  O projeto integra modelagem lógica, aplicação de formas normais (1FN, 2FN, 3FN)
e a implementação de comandos SQL para criação e manipulação de dados.

  O objetivo é demonstrar domínio de SQL (DML), boas práticas de modelagem,
integridade referencial e versionamento utilizando Git e GitHub.
